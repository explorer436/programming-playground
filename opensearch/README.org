How To Return All Documents From An Index In Elasticsearch/OpenSearch?

#+BEGIN_SRC
GET /wikimedia/_search
{
    "query": {
        "match_all": {}
    }
}
#+END_SRC

or

#+BEGIN_SRC
curl -X GET "https://{YOUR_SERVER}:9200/{YOUR_INDEX}_search" -H 'Content-Type: application/json' -d
#+END_SRC
